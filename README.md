# Supply Chain Delay & Cost Optimization Analysis

## Project Overview
This Power BI project analyzes supply chain performance by evaluating shipment delays, transportation costs, and logistics efficiency.

## Objectives
- Identify major causes of shipment delays
- Compare transportation modes and supplier performance
- Analyze logistics costs across routes
- Discover cost optimization opportunities

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
- Data Modeling

## Key KPIs
- Delivery Success Rate
- Average Delay Duration
- Cost per Shipment
- On-Time Delivery Percentage

## Dashboards
1. Executive Overview Dashboard
2. Delay Analysis Dashboard
3. Cost Optimization Dashboard

## Business Impact
Provided actionable recommendations to reduce logistics costs and improve delivery performance through data-driven insights.
